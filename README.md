# Spring Boot + MongoDB CRUD API

## 📌 Project Overview  
This is a RESTful API for managing products using Spring Boot and MongoDB. The API supports CRUD operations:  
- **Create a Product**  
- **Retrieve All Products**  
- **Get Product by ID**  
- **Update Product**  
- **Delete Product**  

## 🚀 Tech Stack  
- Java (Spring Boot)  
- MongoDB  
- Spring Data MongoDB  
- Maven
